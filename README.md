# my-project

---

## 📌 What is this repository

这是 **Songcun Lab 的主资产仓库**。

它不是一个“成品项目”，  
也不是为了展示某个完整系统。

而是一个 **长期持续更新的个人技术与认知账本**，用于记录我如何：

- 学习 AI / 编程 / 自动化
- 使用 AI 解决真实问题
- 在实践中形成可复用的方法与流程
- 将零散经验沉淀为可积累的数字资产

这里的内容可能：
- 不完整
- 不优雅
- 甚至失败

但它们都是真实发生、对我长期有价值的过程。

---

## 🧠 Core Principles

- **长期主义**：记录的是“可持续成长”，而不是一次性成果  
- **过程优先**：过程本身就是资产  
- **真实可追溯**：不美化、不删失败  
- **可复用**：所有内容都以“未来还能不能再用”为判断标准
